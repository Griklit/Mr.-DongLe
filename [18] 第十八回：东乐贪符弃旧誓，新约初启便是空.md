# 第十六回：东乐贪符弃旧誓，新约初启便是空

倏忽半载，时进半岁庆典，郭生偶阅图鉴，忽见维什戴尔之影，忽拊掌哂曰：「斯人已陈旧制，何足萦怀？当俟新岁黍返，以三百符迎之！」未数日，半岁庆典启，银雪双曜临池。郭生睹其锋锐，暗忖：「试投数十符，当无碍大计。」岂料金光乍现，竟得限定银灰。贪焰复炽，自谓：「并纳双英，岂非天意？」遂倾囊百四十符穷追初雪。然天命难测，连得五度银霜，终不见雪影。
检视行囊：玉余五百，源石罄尽，黄青符契早易资财。屈指而计，距维什戴尔再现止仅七日，三百符成镜花水月；岁旦黍池虽有三旬之期，然资财既竭，至岁末勉攒二百符，竟连黍亦难图矣。乃抱膝空对符阵，仰天狂笑：「往岁破誓弃维什戴尔而取浊心斯卡蒂，今岁破誓弃维什戴尔而欲求黍，今日复破誓弃黍而逐新艳，竟致诸计皆空！」

太史公评曰：

```
半载图谋又成空，池迷双曜溃前功。
未得新锋失故剑，再荒陇亩废春耕。
守株犹可待一兔，欲攫双鳞两袖风。
纵使符池千重雪，空使旧笑换新声。
虹光易散琉璃脆，星斗难填玉宇空。
岁岁千符掷虚浪，依旧孤江独钓翁。
明知欲海深千尺，前度沉沦今又迎？
向使专志守黍约，何至临渊泣断缨！
```
